Product · MVP

Chrome Extension for physicians

A browser extension that records the physician's voice during visits, converts it into structured medical data, and automatically fills fields in the medical system.

  • Audio recording via MediaRecorder API
  • Whisper for transcription and GPT‑4o for structuring
  • Auto-filling forms via Content Script
  • Stripe subscription and credit limits

Technical stack

Technologies powering our Chrome Extension

Frontend

  • Chrome Extension (Manifest V3)
  • JavaScript (Vanilla/React)
  • Content Scripts for form interaction
  • MediaRecorder API for audio capture

Backend & Services

  • Node.js + Express server

  • OpenAI API (Whisper + GPT-4o)
  • Stripe for subscriptions
  • Firebase for Google authentication
